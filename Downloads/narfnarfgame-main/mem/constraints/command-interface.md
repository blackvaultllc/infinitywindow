---
name: Command interface constraint
description: Active gameplay must always show the original command interface with map, countries, HUD, powers, and light/shadow choices.
type: constraint
---
Never replace the playable Narf Narf experience with a simplified arena scoreboard, plain lobby, or non-map combat screen.

**Why:** The command interface is the core game perspective: players must see the map, countries, current disaster target, HUD, powers, and human light/shadow choices while playing.

**How to apply:** Any play mode (solo, practice, co-op, ranked/4v4) should route into the command interface flow, not a separate turn scoreboard.
